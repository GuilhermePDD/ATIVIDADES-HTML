HTML (Linguagem de Marcação de HiperTexto)

é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.
Outras tecnologias do HTML geralmente são usadas para descrever a aparência/apresentação (CSS)
ou a funcionalidade/comportamento (JavaScript) de uma página da web.

TAG UTILIDADE
HTML Define o escopo de um documento HTML
P Define um parágrafo no documento
A Tag de Ancoragem, onde podemos inserir
link em elementos de uma página HTML
Head Contém metadados, como o título ou
links para estilos
Body Define o conteúdo visível da página.
2. Etiquetas de metadados (dentro de head)
Fornecem informações sobre o documento.
